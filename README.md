# 中文

## m3u8 下载器

一个 m3u8 下载器
使用 Python 编写，支持 AES-128 解密

### 怎么使用

注意：m3u8 下载器需要搭配 ffmpeg 使用，把 ffmpeg.exe 放在和 run.py 的同级目录下  
ffmpeg 下载链接：https://ffmpeg.org/download.html

```
run.py https://example.com/index.m3u8
```

An m3u8 Downloader.
